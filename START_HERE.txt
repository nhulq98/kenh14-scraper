🔧 STEP 2: INSTALL
═══════════════════════════════════════════════════════════════

   1. Make sure Node.js is installed
      Check: node --version
   
   2. Install dependencies
      $ npm install
   
   3. Wait for ~10 seconds for installation


🚀 STEP 3: RUN
═══════════════════════════════════════════════════════════════

   $ npm start
   
   You should see:
   🚀 Server running at http://localhost:3000
   📰 Kênh14 Scraper is ready!


🌐 STEP 4: USE
═══════════════════════════════════════════════════════════════

   1. Open browser
   2. Go to: http://localhost:3000
   3. Paste Kenh14 URL
   4. Click "Lấy Nội Dung"
   5. See content → Copy → Done!

❓ COMMON ISSUES
═══════════════════════════════════════════════════════════════

   Port 3000 taken?       → Change PORT in server.js
   npm not found?         → Install Node.js from nodejs.org
   Module not found?      → Run npm install again
   Can't connect?         → Check if npm start is running
   No content?            → Check if URL is from kenh14.vn
