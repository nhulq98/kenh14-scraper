## 📌 Hướng Dẫn Tiếp Theo

### Bước 1: Cấu Hình Viettel Token
1. Đăng ký tài khoản: https://viettelai.vn/
2. Lấy API Token từ dashboard
3. Thêm vào file `.env`:
   ```
   VIETTEL_TTS_TOKEN=your_token_here
   ```

### Bước 2: Chạy Ứng Dụng
```bash
npm start
```

### Bước 3: Truy Cập Web
```
http://localhost:3000
```

### Bước 4: Sử Dụng TTS
- Cuộn xuống mục "🔊 Viettel Text-to-Speech"
- Nhập văn bản, chọn giọng và tốc độ
- Nhấn "🎙️ Chuyển Thành Âm Thanh"
- Nghe thử hoặc tải về MP3
